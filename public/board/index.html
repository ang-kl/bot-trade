Kalman filter — release-readiness confidence

Each gate (tests · types · build · review · deploy · live-QA) is a noisy read of “are we ready to ship?”. The filter fuses them into a confidence estimate (line); the shaded band is uncertainty — it shrinks as gates pass. A real 1-D Kalman filter over the gate sequence.
